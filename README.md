Gestão e Inteligência de Dados Financeiros 
Este projeto é uma solução Full-stack desenvolvida para automatizar a gestão de agendamentos, serviços e cálculos de comissões em estabelecimentos de . 
Ele demonstra habilidades tanto em Engenharia de Dados quanto em Desenvolvimento Web .
#Tecnologias Utilizadas
• Backend: Python 3 + Flask (Arquitetura REST)
• Frontend: HTML5, CSS3 e JavaScript (Vanilla ES6+)
• Processamento de Dados: Lógica de agregação e transformação de dados em Python.
• Arquitetura: Comunicação assíncrona via Fetch API.
#Destaques Técnicos
1. Engenharia e Processamento de Dados
O sistema atua como uma ferramenta de ETL (Extract, Transform, Load) funcional.
• Transformação: No arquivo app.py, a rota /api/relatorio processa dicionários complexos de agendamentos, filtra status e aplica cálculos de comissão variáveis.
• Insights: Converte dados brutos em métricas de Faturamento, Comissões e Lucro Líquido, entregando inteligência de negócio pronta para consumo.
2. Desenvolvimento Web e Automação
• API RESTful: Implementação de métodos CRUD (GET, POST, DELETE, PATCH) para gerenciamento de usuários e serviços.
• Single Page Application (SPA): Interface dinâmica que gerencia o estado da aplicação sem recarregar a página, tratando autenticação de gerente e níveis de acesso.
• Field Test: Estrutura pronta para implementação de testes automatizados de integração.
